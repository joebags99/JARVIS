# Profile (example template)

Copy this file to `profile.md` in the same folder and fill it in with your real
details. `profile.md` is **gitignored** — it never leaves your machine.

    cp context/profile.example.md context/profile.md

JARVIS injects everything below into its system prompt, so write it the way you'd
brief a sharp new assistant on day one.
